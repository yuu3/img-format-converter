# 画像変換・軽量化アプリ 機能一覧

## 最低限必要な機能

### 1. 画像アップロード機能
- ドラッグ&ドロップでの画像アップロード
- ファイル選択ダイアログからのアップロード
- 複数画像の同時アップロード対応
- 対応フォーマット: JPEG, PNG, WebP, GIF

### 2. 画像プレビュー機能
- アップロードした画像の表示
- 変換前後の比較表示
- ファイルサイズの表示（変換前後）

### 3. フォーマット変換機能
- 主要フォーマット間の変換
  - JPEG ⇔ PNG ⇔ WebP
- フォーマット選択UI

### 4. 画像圧縮・軽量化機能
- 品質調整スライダー（0-100%）
- リサイズ機能（幅・高さ指定、または比率指定）
- ファイルサイズの削減率表示

### 5. ダウンロード機能
- 変換後の画像を個別ダウンロード
- 複数画像の一括ダウンロード（ZIP化）

### 6. UI/UX要素
- シンプルで直感的なインターフェース
- 変換進行状況の表示
- エラーハンドリングとエラーメッセージ表示

### 7. ブラウザ上での処理
- クライアントサイドでの画像処理（プライバシー保護）
- Canvas APIまたはWebAssemblyを使用
- サーバーアップロード不要

## 技術的に必要なもの

### ライブラリ・API
- 画像処理ライブラリ（browser-image-compression、sharp等）
- Canvas API
- File API / FileReader API
- JSZip（複数ダウンロード用）

### 技術スタック
- React 19.2.0
- TypeScript 5.9.3
- Vite 7.2.4

## 開発優先順位
1. 画像アップロード機能
2. 画像プレビュー機能
3. フォーマット変換機能
4. 画像圧縮・軽量化機能
5. ダウンロード機能
6. UI/UX改善
7. 複数画像の一括処理機能
